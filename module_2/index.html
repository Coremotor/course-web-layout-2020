Описание курса

Вы познакомитесь со стандартом Web 2.0, научитесь работать с макетами и форматировать код, освоите адаптивную верстку, а по итогу - сможете создавать быстрые и адаптивные сайты, которые точно понравятся пользователям

Вы поймете назначение сновных тегов HTML сможете без усилий стилизовать и размещать элементы с помощью CSS и сверстаете свой перый макет.
